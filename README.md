# 2017杭州云栖大会精华 PPT


杭州云栖大会是阿里集团一年一度的全生态科技盛会。此次大会包含140多场技术主题论坛，共计 800 多个主题分享，涵盖人工智能、金融科技、量子计算、生命科学、IoT、政务、多媒体、VR 等 20 多个前沿科技领域。多位全球顶级学术专家、以及 MySQL & MariaDB 创始人、PostgreSQL 联合创始人、Redis 创始人等大咖都献出了精彩的分享。
